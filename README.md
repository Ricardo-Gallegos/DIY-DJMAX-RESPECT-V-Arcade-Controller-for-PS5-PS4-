# DIY-DJMAX-RESPECT-V-Arcade-Controller-for-PS5-PS4-
A DIY arcade-style controller for DJMAX RESPECT V, built by soldering arcade buttons directly to a PS4 controller motherboard for PS5/PS4 compatibility. Designed for fast, responsive input and an authentic arcade rhythm game feel.
